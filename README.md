## About

I'm **Oussama Mosbah**, a **Front-End Engineer & IT Engineer** with **3+ years of experience** building modern, scalable and production-ready web applications.

I specialize in **React.js**, **TypeScript**, **Next.js** and **Material UI**, with a strong focus on clean UI architecture, responsive interfaces, dashboards, API integrations and performance-oriented front-end development.

Currently, I work as a **Front-End Developer at Astrolab**, where I contribute to business-critical web platforms and scalable front-end systems.
